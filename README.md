# Speed Reader

A lightweight, local speed-reading tool inspired by "Spritz" visual alignment and speed ramping.

## Features
- **Optimal Recognition Point (ORP):** Words are centered on a specific character (highlighted in red) to minimize eye movement.
- **Dynamic Speed Ramping:** Automatically increases speed from your `Start WPM` to `End WPM` over the first 10 seconds of reading.
- **Real-time Control:** A speed slider allows you to manually adjust WPM while reading.
- **Dark UI:** Optimized for focus and eye comfort.

## Keyboard Shortcuts
- **Space:** Play / Pause
- **Right Arrow:** Next word (when paused)
- **Left Arrow:** Previous word (when paused)
- **Esc:** Return to the setup screen

## How to Run
Simply open `index.html` in any modern web browser. No installation or internet connection required.
